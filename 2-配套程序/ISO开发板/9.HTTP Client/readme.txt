例程描述
========
同大多数开源平台一样，Yeelink提供的API也是基于HTTP协议提交和接收数据。
在这个应用代码中，W5500EVB作为http client去连接http服务器Yeelink云平台。
W5500EVB连接简单的温湿度传感器dht11，然后将得到的温度和湿度数据提交到Yeelink云平台上。
具体W5500和Yeelink云平台的建立过程，大家可以上网学习一下。

包含文件目录
============
stm32f10x_conf.h     配置库文件
stm32f10x_it.c       中断头文件
stm32f10x_it.h       stm32f10x_it.c的头文件
main.c               主程序


硬件环境
========
这个例程用于野火STM32F103VET6开发板同以太网芯片W5500搭建的应用平台，也可以很方便地
移植到其他平台上。

注意: 
首先通过串口线连接PC和W5500模块；如果W5500模块直接通过网线和PC相连接，需要修改PC的IP为静态IP，且保证和W5500在同一个网段；
如果W5500模块直接连接路由器，则不需要修改。

操作步骤
========
1. 成功编译应用程序，然后通过串口烧录工具或者Jlink把程序烧录到开发板中，dht11接到开发板的正确位置
2. 打开串口工具并复位开发板，如果温度和湿度获取正常，可以看到串口在不停打印得到的温湿度
3. 登陆Yeelink云平台账号，打开 我的设备》管理设备。如果温湿度成功上传Yeelink云平台，可以发现网页中曲线发生变化，并且与串口得到		信息一致
4. Yeelink云平台也提供了一个手机app，可以通过手机观察到此信息
